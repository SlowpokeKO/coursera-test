Anagram: A Coding Project

Given two strings write a function that checks if they are anagrams. The function should return true or false. Display a message indicating the result of the function.

Note: An anagram is a word or phrase formed by rearranging the letters of a different word or phrase.

Examples:
  • listen = silent
  • William Shakespeare = I am a weakish speller
  • eleven plus two = twelve plus one
  • a gentleman = elegant man
  • Apple = paple
